# 双指针技巧

对于数组和链表数据结构，双指针技巧基本上是经常用到的。双指针主要分为两类: **左右指针**和**快慢指针**。

一般来说，左右指针就是两个指针相向而行或者是相背而行。 快慢指针是两个指针同向而行，只是一个快、一个慢.
